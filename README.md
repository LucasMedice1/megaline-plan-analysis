# Megaline Plan Analysis

## Project Overview

This project analyzes customer behavior and revenue generation for Megaline's two prepaid plans: Surf and Ultimate.

The goal is to determine which plan generates more revenue and provide insights that can support marketing and business decisions.

## Skills Demonstrated

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Data Visualization
- Revenue Analysis
- Pandas
- NumPy
- SciPy
- Matplotlib
- Seaborn

## Business Question

Which prepaid plan generates more revenue for Megaline: Surf or Ultimate?

## Project Files

- `megaline-plan-analysis.ipynb` – Complete project notebook containing data preparation, analysis, visualizations, statistical tests, and conclusions.
